# AI智能胸牌 · SaaS管理平台

> 面向多行业企业的AI销售赋能SaaS后台管理系统。核心能力：录音设备管理 → AI智能听记分析 → 销售能力提升 → 金牌销售画像沉淀 → 话术库建设。

## 🚀 功能模块

### 已有功能
| 模块 | 说明 |
|------|------|
| 📊 数据概览 | 客户/评分/设备核心指标，趋势图，排行榜 |
| 📱 设备管理 | 设备列表、详情、屏幕模板、工作时间配置 |
| 🎙️ AI听记 | 录音列表、AI总结报告、评价模板、评价策略、产品卖点 |
| 🎯 线索管理 | 线索跟踪、意向等级、转化漏斗 |
| 👥 系统管理 | 部门/员工管理、角色权限、权限点管理 |

### 🆕 新增核心模块
| 模块 | 说明 |
|------|------|
| 🏭 **行业管理** | 7大行业（银行/汽车/医疗/教育/地产/零售/保险）独立配置 |
| 🤖 **行业AI分析** | 每个行业专属AI分析模板、提示词、评价维度 |
| 🏢 **企业管理（增强）** | 创建企业时选择行业 + AI配额管理（可打开AI总结页面数量） |
| 🏆 **金牌销售画像** | AI自动生成销售画像、能力雷达图、沟通风格标签、优秀案例 |
| 💬 **话术库** | 行业/场景/阶段多维分类，效果评分，AI优化建议，真实案例引用 |

## 🏗️ 技术栈

- **Vue 3** + Composition API
- **Element Plus** UI组件库
- **Vue Router 4** 路由管理
- **Pinia** 状态管理
- **ECharts** 数据可视化
- **Vite** 构建工具

## 📦 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📁 项目结构

```
src/
├── layout/          # 主布局（侧边栏+顶部导航）
├── router/          # 路由配置
├── stores/          # Pinia状态管理
├── views/
│   ├── dashboard/   # 数据概览
│   ├── device/      # 设备管理
│   ├── audio/       # AI听记
│   ├── clue/        # 线索管理
│   ├── sales/       # 金牌销售（画像+话术库）
│   ├── enterprise/  # 企业管理（含行业选择+AI配额）
│   ├── industry/    # 行业管理（配置中心）
│   ├── system/      # 系统管理（员工/角色/权限）
│   └── login/       # 登录页
```
